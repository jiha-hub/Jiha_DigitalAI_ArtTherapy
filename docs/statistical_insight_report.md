# 📊 마인드팔레트: 데이터 분석 및 통찰 전략 (Analytical Insights)

분석 에이전트(간호사+데이터 전문가)와 DB 관리자가 협업하여 사용자 및 치료자에게 제공하는 핵심 통찰 리포트 전략입니다.

## 1. 사용자(환자)에게 제공하는 가치
사용자 스스로 자신의 정서 상태를 인지하고 조절할 수 있는 **'자기 객관화'** 지표를 제공합니다.

- **정서적 컬러 스펙트럼 (Emotional Color Palette)**: 
    - 기간별 가장 많이 사용된 색상을 분석. (예: "지난주에는 차가운 파란색 계열이 많았으나, 이번 주에는 따뜻한 노란색의 비중이 30% 증가했습니다.")
- **심박수-드로잉 동기화 리포트 (Bio-Art Analysis)**: 
    - 심박수가 높을 때와 낮을 때의 드로잉 패턴(필압, 선의 복잡도) 차이를 분석하여 스트레스 해소 시점을 시각적으로 확인시켜 줍니다.
- **자기 효능감 지수 (Self-Efficacy Index)**: 
    - 인페인팅 수정을 통해 이미지를 긍정적으로 변화시킨 횟수와 만족도를 수치화하여 "내가 나의 환경(그림)을 변화시킬 수 있다"는 성취감을 고취합니다.

## 2. 치료자(상담사)에게 제공하는 가치
상담사가 환자의 내면을 더 깊이 이해하고 과학적 근거에 기반한 상담을 진행할 수 있도록 돕습니다.

- **B-H-P (Back-Heart-Prompt) 통합 분석**: 
    - 배경 스케치(구도), 심박수(생체 신호), 프롬프트 키워드(언어)를 교차 분석하여 잠재된 심리적 결핍이나 트라우마 징후를 조기에 포착합니다.
- **위험 단어/패턴 시계열 분석**: 
    - 일기 내 위험 단어 빈도 변화를 추적하여 고위험군의 급격한 정서 변화를 알림으로 제공합니다.
- **선호 테마 분석 (Therapeutic Preference)**: 
    - 10대 테마 중 환자가 가장 많이 선택하는 항목(예: '방해 요소 제거' vs '상징적 개체 추가')을 통해 환자의 주된 방어 기제나 욕구를 파악합니다.

## 3. 데이터 관리 및 보안 (DB Admin 역할)
- **익명화 통계 데이터베이스**: 모든 사용자의 데이터를 익명화하여 '연령대별 평균 정서 흐름' 등의 거대 트렌드를 생성, 서비스 고도화의 근거로 활용합니다.
- **하이브리드 보안 저장**: 민감한 생체 데이터와 일기는 로컬에 우선 저장하되, 상담 연결 시에만 암호화된 터널을 통해 전문가에게 전송되도록 관리합니다.
